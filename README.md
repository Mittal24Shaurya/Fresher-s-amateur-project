# Fresher-s-amateur-project
It is a AI based project which we made for institute competition(IIIT Dharwad).Using this project we won first prize among 35 teams.
These all technologies are used by AI to make dashboard.
Frontend:
├── React 19 (UI)
├── React Router DOM 7 (Routing)
├── Recharts 3 (Charts)
├── Axios (HTTP)
├── Lucide React (Icons)
└── Custom CSS3 (Styling)

Backend:
├── FastAPI (Framework)
├── Pandas (Data Analysis)
├── NumPy (Math)
├── Motor (MongoDB Driver)
├── Pydantic (Validation)
└── Uvicorn (ASGI Server)

Infrastructure:
├── MongoDB (Database)
├── Supervisor (Process Manager)
├── Yarn (Package Manager)
└── Environment Variables (Config)

Design:
├── Google Fonts (Typography)
├── Glass-morphism (Effects)
└── Gradient System (Colors)
